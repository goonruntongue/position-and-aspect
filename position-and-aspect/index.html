子要素のpadding-topの%指定は親要素の横幅に従う

ブロック要素の縦横比を固定するCSSハック

コード 
div.container{
    position:relative;
}
コード 
div.container img{
    position:absolute;
}
コード 
div.container::after{
    content:"";
    display:block;
    width:100%;
}
padding-topの値 0%
コード 
div.container::after{
    content:"";
    display:block;
    width:100%;
    padding-top:0%;
}
コード 
img.base {
    width:100%;
    height:100%;
}
幅 20%
コード 
img.red,img.blue {
    width:20%;
    height:auto;
}
赤丸画像の位置を%で指定します。
0%
0%
コード 
img.red {
    top:0%;
    left:0%;
}
青丸画像の位置を%で指定します。
0%
0%
コード 
img.blue {
    top:0%;
    left:0%;
}
ここまでの設定ができたら、ブラウザを縮めてみましょう。デバイス幅が変わっても位置がずれないポジション指定ができているはずです。
